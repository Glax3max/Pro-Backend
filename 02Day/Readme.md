Day - 02 Learning:

1.Primitive Datatypes (Number,string ,boolean,null,undefined,symbol)
    ↪Non modifiable
    ↪Comparable (two equal primitve give true)

2.Non-Primitive Datatypes (Object,Arrays)
    ↪Modifiable(mutable)
    ↪Not Comparable (two equal non-primitives gives false)
        ⇒ Non-primitive values are referred to as reference  types , because they are being compared by reference instead of value
    ↪Rule to thumb , never compare two non - primitive Data types
        
3.All the Methods of String object
    ↪match
    ↪substring
    ↪startsWtith
    ↪endsWith ... etc

4.Changing Data type (Casting)
    ↪String to Int
        =>parseInt()
        =>Number()
        =>Plus sign(+)
